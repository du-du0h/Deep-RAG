# DeepRAG Web前端

这是DeepRAG项目的Web前端部分，基于React和TypeScript构建。

## 功能

- 用户认证系统（登录/注册）
- 知识库管理（创建、编辑、删除）
- 聊天助手创建和管理
- 基于RAG的智能问答界面

## 技术栈

- React 18
- TypeScript
- React Router
- Ant Design
- Tailwind CSS
- Axios

## 开发指南

### 安装依赖

```bash
npm install
```

### 启动开发服务器

```bash
npm start
```

### 构建生产版本

```bash
npm run build
```

## 项目结构

- `src/components`: 通用组件
- `src/pages`: 页面组件
- `src/assets`: 静态资源文件
- `src/utils`: 工具函数
- `src/services`: API服务
